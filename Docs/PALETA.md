# Paleta de cores

Tokens oficiais usados no frontend (`frontend-prova/src/theme/colors.js` e `index.css`).

| Token | HEX | Uso |
|-------|-----|-----|
| Azul Base | `#3379BC` | Botões primários, sidebar, títulos, links |
| Azul Hover | `#40A5DD` | Hover / estado activo |
| Azul Leve | `#C8E5F4` | Cabeçalhos de tabela, fundo suave |
| Background | `#FFFFFF` | Cards e painéis |
| Background 2 | `#F9FAFA` | Fundo da página, hover de linhas |
| Outline | `#EBEDEE` | Bordas e divisórias |
| Trancado ou Explicativo | `#C8C8C8` | Desactivado / explicativo |
| Texto Placeholder | `#AFAFAF` | Placeholders de input |
| Título de Campo | `#9B9B9B` | Labels / legends |
| Corpo de Texto | `#7A7A7A` | Texto secundário |
| Tooltip | `#525252` | Texto de suporte / botões secundários |
| Texto Destaque | `#000000` | Valores e títulos fortes |

Classes Tailwind correspondentes: `azul-base`, `azul-hover`, `azul-leve`, `fundo`, `fundo-2`, `outline`, `trancado`, `placeholder`, `titulo-campo`, `corpo`, `tooltip`, `destaque`.
